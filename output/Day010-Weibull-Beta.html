💡 今日知识点:Weibull 的 β 参数

Weibull 分布的形状参数 β(Beta)决定了失效模式的类型:

🗣️ 人话版
β 就是告诉你"失效是越来越快还是越来越慢":
• β < 1:越用越不容易坏(早期不良品已经坏了,剩下的越来越安全)
• β = 1:随机器,跟用多久没关系(像抽彩票)
• β > 1:越用越容易坏(磨损、疲劳、老化,用得越久坏得越快)

传音场景里,跌落测试基本都是 β > 1--因为手机越摔越脆弱,这是物理规律。
📊 β 值判断规则
• β < 1 → 早期失效(递减失效率)→ 查来料质量 / 制造缺陷
• β = 1 → 随机失效(恒定失效率)→ 指数分布,外部随机因素
• β > 1 → 磨损失效(递增失效率)→ 查设计 / 工艺问题

四种典型形态(η=10 时):
• β=0.5:最早下降后趋缓(典型早期失效)
• β=1:指数下降(随机失效,分界线)
• β=2:平滑下降(典型磨损失效)
• β=5:悬崖式下坠(快速磨损,寿命很集中)
📱 传音场景:W-795 跌落测试
• MLE 结果:β = 1.98
• β > 1 → 磨损失效型
• 失效根因在于设计 / 工艺(随跌落次数增加,失效率递增)
• 改善方向:优化结构设计、改进工艺
• 数据未显示混合失效模式 → 不需要拆分子群体

⚠️ 避坑指南
• 数据集中在 2~3 个值时,务必确认使用 MLE 而非 LSXY
• LSXY 可能使 β 算崩到 9+(完全失真)
• 如果 β 结果异常大(如 >5)且数据值很集中 → 先检查估计方法
• β 的点估计值要看,同时也要看置信区间--区间太宽说明样本量不够
📝 样本量要求
• β 的估计对样本量敏感,建议至少 15~20 个样品
• 样品太少 → β 的置信区间极宽 → 无法可靠判断失效模式
• 传音内部建议:关键机型 30+ 台
