📌 用途
Kaplan-Meier 用于证明整改是否有效，而非凭感觉判断。
📌 数据准备
C1 = 失效次数，C2 = 删失标记，C3 = 组别（整改前 / 整改后）。
📌 判断标准
Log-Rank p < 0.05 → 整改有统计显著改善，可放行。

💡 今日知识点

Cpk 过程能力分析

Cpk = min((USL − μ) / 3σ, (μ − LSL) / 3σ)

Cpk 衡量的是：制程能力是否满足规格要求。Cpk 越大，说明制程"余量"越足。

💡 理解：Cpk 本质上是"允许的波动"除以"实际的波动"。Cpk=1.33 意味着制程规格带宽是实际波动的 4 倍，所以即使有一点偏移，也不会超出规格限。

【操作】第一步：正态性检验

菜单：统计 → 基本统计量 → 正态性检验

  • 变量 = C1
  • p ≥ 0.05 说明正态 → 可用能力分析

【操作】第二步：能力分析

菜单：统计 → 质量工具 → 能力分析 → 正态

  • 单列选 C1
  • 规格下限填 LSL 值
  • 规格上限填 USL 值

【结果解读】

  • 第一眼看 Cpk 值（≥1.33 合格）
  • 第二眼看直方图上的红线（规格限）有没有被碰到的趋势
  • 第三眼看"整体预期不合格品率"（应该 < 63 PPM）
  • 注意：Cpk 和 Ppk 差距大说明批次间波动大，需追查原因
📱 传音场景：USB 插入力规格 0.5~2.0 Kgf，Cpk=2.14，制程能力充足，不合格品率 < 30 PPM。
