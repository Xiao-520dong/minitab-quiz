📘 Day 009：Kaplan-Meier 整改对比

Minitab 在传音可靠性实验室的应用实战手册 · V1.5

🧠 昨日知识卡片 · Weibull B10 寿命分析

卡片 1：B10 = 总体中 10% 产品失效时对应的时间点。"B10=8次"意味着 10% 的手机在 8 次跌落前会坏。

卡片 2：数据准备：C1=失效次数（全过填测试上限，不能填 0），C2=删失标记（0=失效, 1=全过）。

卡片 3：β 值判断：β<1 早期失效（查来料），β>1 磨损型失效（查设计），β≈1 随机失效。

💡 今日知识点：操作三：整改前后 Kaplan-Meier 对比

【目的】用统计方法证明"整改有效"，而非凭感觉说"好像好一些"。

【作用】整改前后各测一批，画生存曲线，Log-Rank 检验判断显著差异。

【价值产出】整改放行有量化依据；研发部门的整改工作被数据证明有效。

📋 数据准备：
  • C1 = 失效次数
  • C2 = 删失标记（0=失效 / 1=全过）
  • C3 = 组别（整改前 / 整改后）
🔧 Minitab 操作：
菜单：统计 → 可靠性/生存 → 非参数分布分析（右删失）
  • 时间列 = C1
  • 删失列 = C2，删失值 = 1
  • 按变量(B) = C3
  • 图形选项卡勾选"生存图"
  • 结果(R)选项选"对数排秩和 Wilcoxon 统计量"
📊 结果解读：
  • Log-Rank 检验 p < 0.05 → 两条曲线有显著差异 → 整改有效，可以放行
  • 同时看生存图中位数（曲线降到 50% 对应的 X 值）：整改后中位数应该明显大于整改前
