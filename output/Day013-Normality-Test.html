🧠 昨日知识卡片 · Weibull 形状参数 β

卡片 1：β<1 = 早期失效（查来料），β≈1 = 随机失效，β>1 = 磨损失效（查设计），β>>1 = 悬崖式失效。
卡片 2：β 是 Weibull 分析里最有价值的一个数——它直接告诉你失效是"天生缺陷"还是"用坏的"。
卡片 3：数据集中在 2~3 个值时，务必改用 MLE 估计，否则 LSXY 会把 β 算崩到 9+。

💡 今日知识点：正态性检验

跑 Cpk 前先检验数据是否正态。p<0.05 → 非正态 → 改用非正态能力分析。

【理解】

Cpk 分析的前提是数据服从正态分布。如果数据不正态，算出来的 Cpk 值可能完全失真。

⚠️ 核心原则：先验证正态性，再计算 Cpk。跳过正态性检验直接算 Cpk 是实验室最常见的错误之一。

【操作】

菜单：统计 → 基本统计量 → 正态性检验
  - 变量 = C1
  - 确定

【结果解读】

✅ p ≥ 0.05 → 正态，可接着跑 Cpk
❌ p < 0.05 → 非正态，改跑「统计 → 质量工具 → 非正态能力分析」
📊 概率图判断：数据点贴合对角线 = 正态；数据点明显弯曲偏离 = 非正态

【传音场景】

USB 插入力数据（30 点）
正态性检验结果：AD = 0.573，P = 0.125 ≥ 0.05 → 正态，可做 Cpk
